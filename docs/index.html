Pokédex

Trainers

TL;DR Changes

  • All trainer battles are Double Battles (except first rival battle)
  • Non-legendary Pokémon Gens 1-9 are available in the game and fully rebalanced.
  • Mega evolution and alternate forms (for the most part) are available - check Mahogany Town
  • This is a difficulty hack but it's not intended to be impossible
  • Rage Candy Bar guy no longer stops you from going towards Route 44

Pokémon Changes

  • Includes type changes, ability changes, and role changes
  • Much better Pokémon availability. Wiki contains location information for each. All should be available before the Elite 4
  • More Eviolite Pokémon should be viable
  • Refer to the wiki for specifics but every non-legendary Pokémon has been rebalanced. The goal is for all Pokémon to be viable (even outside of Mega Evolution) so many have received significant buffs

Item Changes

  • Held items (except berries) are returned to you at the end of the battle
  • Cannot use healing items in battle (held items still work)
  • Poké Marts are the primary way in which new items are added to this hack so check them out in each town
  • Nature-changing mints are available throughout the game in Poké Marts
  • Eviolite prevents regular evolution

EVs/IVs Changes

  • IVs are "lucky" meaning you will most likely end up with 30 IVs in each stat though 0 and 31 are still possible and as likely as they are in vanilla
  • All Pokémon grant 2 EVs in all stats
  • EV/IV viewer in the Pokémon stats panel in-game by pressing L or R

Move Changes

  • Re-usable TMs
  • Forgettable HMs
  • Most moves have been rebalanced. Typically reducing RNG elements
  • Evasion increases are no longer present in the game (though accuracy drops still exist)
